# Unsere-Analysepakete

Die angezeigten Analysepakete, werden aus dem Angaben im backend generiert. So das die entsprechenden Texte über all auf der Website abgeglichen sind. Die Mehrsprachigkeit ist gegeben, wenn das backend vollständig gepflegt ist. Netto-Preise werden nur ausgegeben, wenn der Benutzer angemeldet ist und die Berechtigung hat Preise einzusehen.
