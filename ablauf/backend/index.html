# Matrix (Artikel/Tests)

Um bei dem Prozess für das Beauftragen einer Analyse den Preis zu steuern, wird eine Matrix benötigt. Nach der Auswahl des Artikel wird geprüft ob dieser eine Matrix hat. Die Matrix bietet die Möglichkeit weitere Tests zu beauftragen. In Abhängigkeit zum Artikel, zur Schaummittelart und der Wasserart gibt es unterschiedliche Tests zur Auswahl.

Jeder Test kann als Basistest (nicht abwählbar), Zusatztest oder nicht machen gekennzeichnet werden.

Einige Test sind als Basistest oder Zusatztest gekennzeichnet, diese Test können zur Abwahl gesperrt werden, d.h. Diese sind Notwendig und sind nicht austauschbar.
